# ATM Interface (Java)

## Description
This is a **Java console-based ATM Interface project**.
It allows users to log in using a User ID and PIN and perform basic ATM operations
such as checking transaction history, withdrawing money, depositing money,
transferring funds, and quitting the system.

## Features
- User authentication (User ID & PIN)
- Transaction History
- Withdraw money
- Deposit money
- Transfer money
- Display current balance
- Menu-driven program

## Technologies Used
- Java
- Scanner class
- ArrayList
